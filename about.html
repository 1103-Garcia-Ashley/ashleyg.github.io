List and define any new or unfamiliar terms:

Content Management System (CMS): CMS is a software that is used to control the creation and manipulation of digital content. It is used for enterprise content management (ECM) and web content management (WCM). ECM supports multiple users collaborating one environemnt and WCM is the collaborative authroing for websites that include texts, graphics, photos/videos, or any content that the user can interact with. This said, ECM and WCM usually go hand in hand.

Jekyll: Jekyll is able to create a complete and static website. It is able to dot this because Jekyll has a built in support for Github pages and has a simplified build process which makes it easier to utilize. Additionally, Jekyll also supports Markdown and Liquid which both are capable of loading dynamic content on ones site.
